# 🌐 Nexora - Responsive Landing Page

A modern and responsive landing page built using **HTML**, **CSS**, and **JavaScript**. The project features a fixed interactive navigation bar, smooth scrolling, hover effects, and a clean glassmorphism-inspired design. It was developed as part of the **SkillCraft Technology Web Development Internship – Task 01**.

## 🚀 Features
- Fixed Navigation Bar
- Interactive Navigation Menu
- Navbar Changes on Scroll
- Hover Effects on Navigation Links
- Smooth Scrolling
- Responsive Design
- Mobile Hamburger Menu
- Hero Section
- About Section
- Services Section
- Portfolio Section
- Contact Section
- Scroll-to-Top Button
- Scroll Reveal Animations
- Glassmorphism UI

## 🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript
